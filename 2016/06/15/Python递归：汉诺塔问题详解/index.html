Python递归：汉诺塔问题详解

  • Hanoi
  • python
python

传说印度某间寺院有三根柱子，上串64个金盘。寺院里的僧侣依照一个古老的预言，以上述规则移动这些盘子；预言说当这些盘子移动完毕，世界就会灭亡。这个传说叫做梵天寺之塔问题(Tower of Brahma puzzle)。

比如：我们现在定义：Hanoi(n,A,B,C)，其中n是圆盘的数量，A，B，C是三根柱子：

  • 当n = 1时：A –> C
  • 当n = 2时：A –> B , A –> C , B –> C
  • 当n = 3时：A –> C , A –> B , C –> B , A –> C , B –> A , B –> C , A –> C

如此，我们很容易发现其中的规律：

  • n = 1时是固定的。
  • 从n = 2开始，中间的一个都是A-->C，然后前面都可以归结到一个 A-->B，后面都可以归结到一个 B-->C

所以我们在写程序时完全可以按照这个思路：

def Hanoi(n,a,b,c):
    if n == 1:
        print(a,"-->",c)
    else:
        Hanoi(n-1,a,c,b) #当n=2时，这里就是Hanoi(1,a,c,b),则打印从a->b
        Hanoi(1,a,b,c) #这是中间万年不变的a->c
        Hanoi(n-1,b,a,c) #当n=2时，这里就是Hanoi(1,b,a,c),则打印从b->c

Hanoi(int(input("Please input the disk's no : ")),'A','B','C')

当我们输入的值为2时：

Please input the disk's no ：2
A -> B
A -> C
B -> C

当我们输入的值为3时：

Please input the disk's no ：3
A -> C
A -> B
C -> B
A -> C
B -> A
B -> C
A -> C

验证无误。
