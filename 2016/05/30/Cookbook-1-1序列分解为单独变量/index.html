将序列分解为单独的变量

问题：我们有一个包含N的元素的元组或序列，现在想将它分解成N的单独的变量。

任何序列我们都可以使用一个简单的赋值操作分解成单独的变量，但是要注意变量的总数和结构要与序列一致。

L = {"zhangsan","lisi",'wangwu'}
print(L)
for name in L:
    print("name: %s" % name)
print("----------------")
p = {4,5}
x,y = p
print("p =",p)
print("x =",x)
print("y+x =",x+y)
print("-----------------")
s = "Hello"
a,b,c,d,_ = s
print(a)
print(d)

若是元素数量不匹配会出现错误提示：

p = (4,5)
x,y,z = p

Traceback (most recent call last):
      File "test.py", line 24, in <module>
          x,y,z = p
ValueError: not enough values to unpack (expected 3, got 2)
