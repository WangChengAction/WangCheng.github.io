今天我们尝试爬取租房网站小猪短租的一些信息，小猪短租网站的大体构成如下图所示网站的左侧是房源的实时地图，右侧是房源链接：

我们可以看到，网站右边是由一个个的房源的链接图组成，我们大体观察了下，共有13页，每页显示了24个房源信息。而我们需要做的，就是爬取这13页上所有房源信息的详情，详情页面具体信息如下图所示：

目标

爬取300个房源的具体信息，包括标题，房源地址，日租金，首页图片地址，房东头像地址，房东姓名和房东性别。

程序代码如下：

from bs4 import BeautifulSoup
import requests
import time


url = "http://bj.xiaozhu.com/"
urls = ["http://bj.xiaozhu.com/search-duanzufang-p{}-0/".format(str(i)) for i in range(0,14)]
count = 0

#用来爬取每个详情页的信息的函数   
def collect_info(url,data=None):
    response = requests.get(url)
    soup = BeautifulSoup(response.text,'lxml')
    time.sleep(2)

    #标题，地址，日租金，第一张图片链接，房东图片链接，房东性别，房东名字
    titles = soup.select('body > div.wrap.clearfix.con_bg > div.con_l > div.pho_info > h4 > em')
    addresses = soup.select('body > div.wrap.clearfix.con_bg > div.con_l > div.pho_info > p > span.pr5')
    day_prices = soup.select('div.day_l')
    images = soup.select('div.pho_show_l > div > div:nth-of-type(2) > img')
    host_imgs = soup.select('div.js_box.clearfix > div.member_pic > a > img')
    host_sexes = soup.select('div.js_box.clearfix > div.w_240 > h6 > span')
    host_names = soup.select('div.js_box.clearfix > div.w_240 > h6 > a')

    for title,address,day_price,image,host_img,host_sex,host_name in zip(titles,addresses,day_prices,images,host_imgs,host_sexes,host_names):

        #判断性别
        sex = ""
        for s in host_sex.get('class'): 
            if s == 'member_girl_ico':
                sex = "女士"
            elif s == 'member_boy_ico':
                sex = "先生"
            else:
                sex = "未知"

        #日租金货币符号与钱数连接
        str =''
        price = list(day_price.stripped_strings)
        price = str.join(price)


        data = {
                '标题' : title.get_text(),
                '地址' : address.get_text().strip(),#地址去除多于的空格和回车
                '日租金' : price,
                '图片地址' : image.get('src'),
                '房东头像' : host_img.get('src'),
                '房东性别' : sex,
                '房东姓名' : host_name.get('title')
                }
        print( '标题：' + data['标题'] + '\n' +'图片地址：' + data['图片地址'] + '\n' + '地址 ：' + data['地址']+ '\n' + '日租金 ：' + data['日租金']+ '\n' + '房东 ：' + data['房东姓名']+ '\n' + '房东头像 ：' + data['房东头像']+ '\n' + '房东性别 ：' + data['房东性别'] )
        print("======================")


#获得每一页上所有的房源链接
def find_link(url,data=None):

    global count
    response = requests.get(url)
    soup = BeautifulSoup(response.text,'lxml')
    link = soup.select('ul > li > a[class="resule_img_a"]')

    for l in link:
        if count < 300: #我们需要统计300个房源信息，所以在这里要做限制
            url_pages = l.get('href')
            collect_info(url_pages)
            count = count +1


for url in urls:
    find_link(url)

总结

在爬取过程中，主要遇到以下的几个问题：

房源详情页问题

这一块主要的困难在与房东性别的判断，通过观察我们发现在房东姓名的代码后面，有这么一段代码：<span class="member_boy_ico"></span>，所以我们可以通过这样一段代码来判断房东的性别：

sex = ""
for s in host_sex.get('class'):
    if s == 'member_girl_ico':
        sex = "女士"
    elif s == 'member_boy_ico':
        sex = "先生"
    else:
        sex = "未知"

这里若是房东本人最初并未填写性别，span处为空，性别将会显示未知。

爬取所有房源链接问题

这部分的难点在于确定房源的链接，link = soup.select('ul > li > a[class="resule_img_a"]')这里为了避免爬取到页面头部一些无效的链接，我们需要给a限定class属性的内容，这样才能只爬取到房源的链接。

爬取所有页面

观察每一页的链接：

http://bj.xiaozhu.com/search-duanzufang-p2-0/
http://bj.xiaozhu.com/search-duanzufang-p3-0/
...

http://bj.xiaozhu.com/search-duanzufang-p13-0/

我们发现每个连接只有最后p后的数字不一样，第几页就是数字几，所以我们可以做一个链接集合：

urls = ["http://bj.xiaozhu.com/search-duanzufang-p{}-0/".format(str(i)) for i in range(0,14)]

这样用format中的内容替换{}中的内容，即可完成全部页面的url采集。
