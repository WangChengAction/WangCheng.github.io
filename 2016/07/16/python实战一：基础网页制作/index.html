python实战一：基础网页制作

  • html
  • python
文章目录
  1. 相对路径和绝对路径

实战第一天，就从一个简单的网页开始吧。
成果图大致如下：

首先我们观察下这个网页，主要由头部，中间内容和底部的版权声明三个部分组成，所以这里我们需要三个div将其分成三块:

  • 头部是第一块，其中又包括标题Logo和菜单，标题可以通过图片的导入进行设计，而菜单栏是由列表构成。
  • 中间内容首先是一个标题，然后做一个下划线，接着导入三张图片，一段文本。
  • 最后是版权声明部分，直接用p标签将版权符号导入即可。

网页代码如下:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>The Blah</title>
    <link rel="stylesheet" href="homework.css">
    </head>
    <body>
        <div class='header'>
        <img src="images/blah.png" alt="The logo">
        <ul class='nav'>
            <li><a href="#">Home</a></li>
            <li><a href="#">Site</a></li>
            <li><a href="#">Other</a></li>
        </ul>
        </div>
        <div class='main-content'>

            <hr>

                <li>
                    <img src="images/0001.jpg" alt="The beach" width="150" height='150'>
                </li>

                    <img src="images/0002.jpg" alt="Beautiful girl" width='150' height='150'>
                </li>

                    <img src="images/0003.jpg" alt="sea" width='150' height='150'>               
                </li>
            </ul>
            <p>

相对路径和绝对路径

相对路径是相对于当前文件的路径，网页中一般表示路径都使用这种方法。.是代表当前目录，。。是代表上一级目录
绝对路径是主页上文件或目录从顶级目录、盘符或域名开始的路径。
