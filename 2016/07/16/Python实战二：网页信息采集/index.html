Python实战二：网页信息采集

  • combat
  • html
  • python
文章目录
  1. 总结

了解了网页的大致构成，接下来我们看如何进行页面信息的采集，这个我们主要是采用BeautifulSoup对网页进行解析。类似下面的网页：

我们将其中的图片地址，价格，标题，评分和评分数量一一进行解析。

代码如下：

from bs4 import BeautifulSoup

with open("'/Users/wangcheng/Desktop/Python_Combat/week1/1_2/1_2answer_of_homework/1_2_homework_required/index.html','r'") as html:

    Soup = BeautifulSoup(html,'lxml')

    titles = Soup.select('body > div > div > div.col-md-9 > div > div > div > img')
    images = Soup.select('body > div > div > div.col-md-9 > div > div > div > img')
    prices = Soup.select('body > div > div > div.col-md-9 > div > div > div > div.caption > h4.pull-right')
    reviews = Soup.select('body > div > div > div.col-md-9 > div > div > div > div.ratings > p.pull-right')
    stars = Soup.select('body > div > div > div.col-md-9 > div > div > div > div.ratings > p:nth-of-type(2)')

    for title,image,price,review,star in zip(titles,images,prices,reviews,stars):

        data = {
            'title' : title.get_text(),
            'image' : image.get('src'),
            'price' : price.get_text(),
            'review' : review.get_text(),
            'star' : len(star.find_all('span',class_="glyphicon glyphicon-star"))
        }
    print(data)

其实在里面的难点主要在于星星数目的统计，这里我卡了好久，开始想着用find_all查找"glyphicon glyphicon-star"，每查找到一次，count的数目加一，然后最后输出count的总数目就是星星的数目，但是总是失败，这是由于find_all返回的是个列表，并不是单独的一行行数据，所以我用count+1并没有办法得到预想的答案。

然后我尝试找有没有类似“counter”的函数，能统计find_all查询到的元素的个数，但是没有找到，最后突然看到到文档中find_all返回的是一个列表，我只需要用一个len，就可以知道列表的元素个数，从而就可以计算出星星的数目。

当然，如果我们想要搜集所有star数多于3的图片，可以将所有内容放入一个list里面，然后进行信息的比对，如下代码：

from bs4 import BeautifulSoup

info = []

with open("'/Users/wangcheng/Desktop/Python_Combat/week1/1_2/1_2answer_of_homework/1_2_homework_required/index.html','r'") as html:

    Soup = BeautifulSoup(html,'lxml')

    titles = Soup.select('body > div > div > div.col-md-9 > div > div > div > img')
    images = Soup.select('body > div > div > div.col-md-9 > div > div > div > img')
    prices = Soup.select('body > div > div > div.col-md-9 > div > div > div > div.caption > h4.pull-right')
    reviews = Soup.select('body > div > div > div.col-md-9 > div > div > div > div.ratings > p.pull-right')
    stars = Soup.select('body > div > div > div.col-md-9 > div > div > div > div.ratings > p:nth-of-type(2)')

    for title,image,price,review,star in zip(titles,images,prices,reviews,stars):

        data = {
            'title' : title.get_text(),
            'image' : image.get('src'),
            'price' : price.get_text(),
            'review' : review.get_text(),
            'star' : len(star.find_all('span',class_="glyphicon glyphicon-star"))
        }
    info.append(data) 

    for i in info:
        if float(i['star'])>3:
            print(i['title'],i['price'])

这样即可选出所有评星大于3的图片的标题和价格，info中的内容其实就是将上面例子的结果放入一个列表之中，而i就代表每一行的内容，其最后的结果显示如下：

EarPod $24.99
New Pocket $64.99
New sunglasses $74.99
iphone gamepad $94.99
Best Bed $214.5
iWatch $500
Park tickets $15.5

注：之前例子中有个stripped_string的用法，它是获得一个父标签下所有子标签的文本信息，相当于一个高级的get_text版本。
例如，我们这个文章的标签，标签有两个，用get_text只能获取一个，要想获取全部的标签，就需要使用这个：

'category': list(cate.stripped_strings)

然后将其放入一个list中。

总结

这次我们解析的这个网页是一个本地网页，所以我们采用了open对网页的绝对地址进行了读取，然后用BeautifulSoup对网页进行解析。
解析了整个网页之后，因为我们想要提取网页中所有图片的图片地址，标题，商品价格，评论数和星级等诸多信息，所以我们要对这些信息进行筛选。这里我们主要借助于Google Chrome的检查功能，选中将要爬取的信息的CSS SELECTOR，进行复制，然后根据实际情况做一些修改，筛选出所需要的信息之后，我们需要将这些信息放入一个字典里边，便于进行统计查找，这时可以借助zip()。

zip([seql, …])接受一系列可迭代对象作为参数，将对象中对应的元素打包成一个个tuple（元组），然后返回由这些tuples组成的list（列表）。若传入参数的长度不等，则返回list的长度和参数中长度最短的对象相同。

比如：

>> z1 = [1,2,3]
>> z2 = [4,5,6]
>> result = zip(z1,z2)
>> result
[(1,4),(2,5),(3,6)]

zip()配合*号操作符，可以将已经zip过的列表对象解压：

>> zip(*result)
[(1,2,3),(4,5,6)]

将各种信息放入zip中后，zip(titles,images,prices,reviews,stars)形成了一个列表，里面是一个一个的tuple，然后我们用data创建一个字典，然后将采集到的信息的内容传入这个字典，并进行打印，注意，image我们要获取的是图片地址，所以要用image.get('src')获取图片的地址属性，而star我们要获取的是星星的个数，从网页源码我们可以看到，若有星星，则span的class_="glyphicon glyphicon-star",若没有星星，则是class_="glyphicon glyphicon-star-empty",所以我们可以star.find_all('span',class_="glyphicon glyphicon-star"))找到所有的有星星的情况，此时find_all返回的是一个列表，我们可以用len统计列表中元素的个数，即为星星的数目。

当我们想要做一个查询，查询所有星级大于3的图片的题目和标签，我们可以将之前data中的信息append到一个空的列表中，然后用以下代码进行比对：

for i in info:
    if float(i['star'])>3:
        print(i['title'],i['price'])

这样即可获得最终的结果。
